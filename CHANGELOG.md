# Changelog

## 1.0.0 - Initial GitHub Release

- Added Expo React Native mobile app structure.
- Added Dashboard, Control, Voice, History and Schedule screens.
- Added ESP32 HTTP/WebSocket integration layer.
- Added backend voice/history API services.
- Added mock fallback data and services.
- Added gas warning UI.
- Added default light schedules:
  - 05:00 living room light on.
  - 06:00 bedroom light on.
  - 18:00 all lights on.
  - 22:00 all lights off.
- Added README, WBS and project documentation.
